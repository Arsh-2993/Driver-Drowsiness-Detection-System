#  Driver Drowsiness Detection System

An AI-based Driver Drowsiness Detection System developed using Python, OpenCV, TensorFlow, CNN, and Dlib. The system monitors the driver's eyes and yawning in real time using a webcam. If drowsiness is detected, it triggers an alarm to help prevent accidents.

---

##  Features

- Real-time face detection
- Eye Aspect Ratio (EAR) calculation
- Yawning detection
- AI-based drowsiness prediction
- Alarm notification when driver is sleepy
- User-friendly GUI
